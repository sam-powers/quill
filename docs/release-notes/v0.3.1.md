# Quill v0.3.1

A small patch release — no new features, no document or sidecar format changes.

## Fixed

- **`@claude` always uses your Claude login** — if Quill was launched from a terminal that had an `ANTHROPIC_API_KEY` exported, the Claude CLI it spawns could silently bill that API key instead of your signed-in Claude account. Quill now strips the key from the CLI's environment, so replies always run under your own `claude` login.

## Docs

- The README now opens with a step-by-step **Getting started** section (which `.dmg` to download, drag-to-Applications, the one-time Gatekeeper right-click → Open), and both it and the [User Guide](https://github.com/sam-powers/quill/blob/main/docs/USER_GUIDE.md) explain the in-app update notification added in v0.3.0.

## Install

| Platform              | Asset           |
| --------------------- | --------------- |
| macOS (Apple Silicon) | `…_aarch64.dmg` |
| macOS (Intel)         | `…_x64.dmg`     |

Builds are **unsigned**: right-click **Quill.app** → **Open** → **Open** on first launch.

Updating from v0.3.0? Download the new `.dmg` and replace the app in Applications — your documents are untouched.
